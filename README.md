# Lisar Connect

Lisar Connect helps users create secure VPN connection profiles for standard clients such as OpenVPN and compatible L2TP/IPsec setup.

No mandatory proprietary app.

## What Lisar provides

- VPN connection profiles
- OpenVPN `.ovpn` file download
- Compatible L2TP/IPsec setup values
- Entry and exit routing options
- Active VPN session visibility
- Connection history
- Dashboard-based profile management

## Positioning

Lisar is not trying to replace OpenVPN or native VPN clients.

The goal is to remove VPN vendor lock-in around the service layer: profiles, routing, credentials, active sessions, and setup visibility.
